Intro

Olá, sou Tadeu Tovar, um entusiasta do desenvolvimento web que está agora se aventurando no mundo da ciência de dados. Comecei minha jornada no desenvolvimento web com o curso do Le Wagon e atualmente estou cursando bacharelado em Ciência de Dados na IBMEC-RJ.

Meu portfólio apresenta uma variedade de projetos nos quais apliquei minhas habilidades em tecnologias como Rails, Python, SQL e React. Estou animado para compartilhar meu trabalho e discutir como posso contribuir para seus projetos.

projetos


Caça ao Tesouro

Um projeto em Python para a linha de comando que simula uma caça ao tesouro, utilizando classes, objetos e funções para criar uma experiência interativa e envolvente de busca por riquezas escondidas.

Airbnb Clone

Um projeto colaborativo em Ruby on Rails, baseado no Airbnb, que emprega o uso de banco de dados SQL, Ruby, e segue a arquitetura MVC. Utiliza gems como Pundit para controle de acesso e Devise para autenticação de usuários, proporcionando uma experiência completa e segura para os usuários finais.
